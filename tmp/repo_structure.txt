=== KEY ENTRY POINTS ===

1. MAIN PROCESS (Backend):
   - src/main/index.ts
   - Starts Electron app, HTTP server, IPC handlers
   - Manages file system, sessions, projects

2. RENDERER (Frontend):
   - src/renderer/index.tsx
   - React app entry point
   - Communicates with main process via IPC

3. PRELOAD SCRIPT:
   - src/preload/index.ts
   - Security bridge between main & renderer
   - Exposes safe IPC methods to renderer

4. BUILD CONFIG:
   - electron.vite.config.ts
   - Configures Electron + Vite build
   - Defines main/preload/renderer entry points
